Features

Your website includes a number of functional features to enhance the customer experience.

Features included in your site are:

Listing of Service Locations

The system maintains a list of current service locations in the web app "Locations". Postcodes covered are linked back to a locations web app item. Content for each service location is uniquely set within the web app item definition.

Checking Postcode coverage

The system enables visitors to enter their postcode to check on the level of coverage. System looks up the postcodes web app checking on coverage level if poscode is present.

Type 1: Fully covered
Type 2: Fringe coverage
Type 3: No cover at present but possibly coming

Autodetection of current location

The system uses HTML5 geocoding to determine current location of visitor. If on desktop the browser will prompt aking visitor to give permissin to shae their current location. This is displayed by te site

Direct Call to Customer Service Centre

ll references to the phone numbers provide direct call capability. The use of this feature is dircetly aimed at smart phones

Online Booking Form

Visitors amy also register their booking request which raises an email back to DTY callcentre who will ring the client once the call centre is open for bookings

Other forms of contact

  3. Call Me Back - Form to email call centre for call back

Web Apps

Your site includes a number of web apps. These are customised databases holding data that is uesed within your site for controlling content that is displayed within pages or prsenitng information relevant to your site.

The web apps set up for your website are as follows.

Home Slider

This web app  identifies the details of the home page banner images rotated in the slider. You can have as many images as yu like however we suggest that 3 be the mximum. Each "enabled" item will be shown in the lsider which means you can turn banner images on / off as you please.

You simply select the image you wish to use as a banenr image and enter the text to overlay the image displayed.

As with the location web app, you can define t text to be displayed on button that is displayed (this button will invoke a phone call to 1300 30 38 34 by default).

The "Banner Link" field can be used to direct a visitor to a desired destination if they click on the overlay text. If this is an internal destination then you only need enter it as a relative URL otherwise a full URL is required.

Inner Banner

This web app  identifies the details of the single page banner image shown in the top of the page for any page using the template "Default Template - with BG Image". You can have as many images as you like however in the template we choose a single particular image form the list of images available. If you wish to change the image or details we suggest yu simply edit the item already in place.

You simply select the image you wish to use as a banenr image and enter the text to overlay the image displayed.

As with the location web app, you can define t text to be displayed on button that is displayed (this button will invoke a phone call to 1300 30 38 34 by default).

The "Banner Link" field can be used to direct a visitor to a desired destination if they click on the overlay text. If this is an internal destination then you only need enter it as a relative URL otherwise a full URL is required.

Location

This web app is used to hold the content for each service location page. In order to deliver consistency across all location pages yet still provide nique content, these pages have been set up as a web app. One item exists for each location.

The data held on each location in the web app item is:

  • name - name of yoru location. This is displayed on the pages and should be unbique
  • Franchise ID - unique identifier that is not used within the site at this stage in any way
  • Status - set to available / not available as a future feature for controlling how a location is presented. At this stage this feature is not being used
  • Banner image - you select the image to be shown as a banner in the top of the page. the image must be 1920 x 720 px in size
  • Banner text - this is teh text yu wish to display overlaying your bannre image
  • Banner button (not used) at this point simply enter "#'
  • Banner lin (not use) at this point simply enter a '#'
  • Call button text usually consistent use the following format with the <span> tags
    <span>CALL: 1300 30 38 34</span> CALL ANYTIME
    where the Call anytime will show on the 2nd line
  • Contact Number - this is the phone number that will be called if a visitor clicks on the button
  • Introduction - this sia  block of formatted text appearing at the top of yrou page as the first paragraph
  • ITEM ADDRESS - enter these feilds to denote the physical location of the geaographical centre of your location
  • ITEM DESCRIPTION - Enter the content yu wish to display within the page at this particular service location
